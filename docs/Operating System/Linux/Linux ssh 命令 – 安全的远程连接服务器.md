# Linux ssh 命令 – 安全的远程连接服务器

**参考文档：**

- [ssh命令 – 安全的远程连接服务器 – Linux命令大全(手册) (linuxcool.com)](https://www.linuxcool.com/ssh)

---

**语法格式：** ssh [参数] 远程主机

**常用参数：**

| 参数      | 解释                   |
| ----------- | ------------------------ |
| -p <端口> | 指定远程服务器上的端口 |

**参考案例：**

```shell
ssh -p '22' 192.168.1.102  # 指定 ip 端口 连接服务器
```
