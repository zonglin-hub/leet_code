# Linux rz 命令 – 基于 ZModem 协议上传文件到服务器

**参考文档：**

- [rz命令 – 基于ZModem协议上传文件到服务器 – Linux命令大全(手册) (linuxcool.com)](https://www.linuxcool.com/rz)

---

**语法格式：rz** [参数]

**常用参数：**

| 参数 | 解释                     |
| ------ | -------------------------- |
| -y   | 遇到重名的文件直接覆盖<br /> |

**参考案例：**

```sh
# 上传指定的文件到服务器，遇到重名的文件直接覆盖：
rz -p   
```
