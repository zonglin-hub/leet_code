当两个二进制数进行按位异或（XOR）操作时，如果对应位的值相同，则结果为 0；如果对应位的值不同，则结果为 1。

例如，假设有两个二进制数 1010 和 1100。它们按位异或的结果为：

```rust
1010 XOR 
1100 =
0110

```

另外一个例子，假设有三个数字：2、2 和 3。这三个数字的二进制表示分别为：
```rust
2: 0010
2: 0010
3: 0011

```
将它们进行按位异或运算：

```rust
0010 XOR 
0010 XOR 
0011 = 
0001

```

所以函数返回的值为 1，即只出现一次的数字是 3。
